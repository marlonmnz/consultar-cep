# Consultar CEP

## Descrição da aplicaçao

A ideia dessa aplicação é entender o funcionamendo da 'Fetch API', como consultar e consumir uma API externa utilizando o metodo Fetch.

## Como funciona a aplicação?

O usuário irá digitar o CEP desejado e ao dar 'enter' as informações do local (Logradouro, Bairro, Localidade e UF) serão auto-completadas em seus respectivos campos.

## Legal, mas e por "baixo dos panos"

- Utilizei o Bootstrap para criar o layout da aplicação.
- Utilizei Javascript puro para fazer a requisição através da API do _ViaCep_, buscando assim os dados e retornando em seus respectivos campos no client-side da aplicação
